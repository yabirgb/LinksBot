Saving your links for later


You can send a complete url like github.com and the bot will save it for you. It can be mixed with text.

Examples:

  • Walid save the url yabirgb.com
  • yabirgb.com

Also links in forwarded messages will be saved.
To add links to pocket use the sintax:

!p an_url one_tag other_tag

Example: !p yabirgb.es tech dev maths

To configure pocket use the command /pocket and authorize the app from your browser following the instructions.
